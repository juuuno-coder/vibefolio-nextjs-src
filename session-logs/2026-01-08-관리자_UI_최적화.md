# 세션 로그: 관리자 효율화 및 UI 심플 최적화

## 부제: 군더더기를 덜어내고 본질에 집중하는 사용자 경험 설계

**일시:** 2026년 01월 08일  
**상태:** 완료 및 배포

### 1. 직면한 문제 (Problem)

- **메인 배너의 시각적 과부화:** 기존 메인 배너가 카드 형태의 복잡한 스타일과 과도한 애니메이션을 가지고 있어 시각적으로 어수선하고 사용자 집중도를 떨어뜨림.
- **채용/공모전 카드의 비효율적 비율:** 16:9 비율의 가로형 카드는 세로형 포스터가 주를 이루는 공모전 데이터를 표현하기에 부적합하여 이미지가 잘리거나 작게 보이는 문제가 발생함.
- **관리자 운영의 비효율성:** 공지사항과 팝업 광고가 별도의 메뉴로 분리되어 있어 이중 관리가 필요했으며, 팝업 이미지 업로드 기능의 부재로 수동 URL 입력이 필요했음.

### 2. 해결 접근 방식 (Approach)

- **UI 단순화 (Simplification):** "본질이 디자인이다"라는 원칙 아래 배너의 카드 테두리, 그림자, 배경색을 과감히 제거하고 꽉 찬 이미지(Full-bleed)와 깔끔한 타이포그래피를 적용함.
- **데이터 중심 UI 설계:** 공모전 포스터의 특성을 고려하여 카드 비율을 3:4 세로형으로 변경하고, 이미지가 잘리지 않으면서도 전체를 가득 채우는 시각적 구성을 채택함.
- **관리 프로세스 통합:** 팝업 관리를 공지사항의 한 속성(Flag)으로 통합하여 관리 동선을 단축하고 운영 편의성을 극대화함.

### 3. 주요 과제 및 해결 (Challenges & Solutions)

- **가로/세로 이미지 대응:** 가로형 이미지 사용 시 발생하는 여백 문제를 해결하기 위해 엠비언트 블러(Ambient Blur) 배경 효과를 도입했으나, 최종적으로는 확대 애니메이션을 제거하고 이미지를 꽉 차게 보여주는 더 간결한 방식으로 사용자의 최종 승인을 얻음.
- **스마트 정렬 로직 구현:** 단순히 최신순이 아닌, 사용자의 니즈에 맞춰 '마감 임박 공고'를 우선 노출하고 마감된 공고는 리스트 최하단으로 보내는 지능형 정렬 시스템을 적용함.
- **관리자 업로드 기능 보강:** 공지사항 통합 관리 환경에서도 팝업 이미지를 즉시 업로드할 수 있도록 Supabase Storage 연동 버튼을 추가하여 운영 효율을 높임.

### 4. 최종 결과 (Outcomes)

- **메인 배너:** 군더더기 없는 프리미엄 스타일로 전면 개편되어 메시지 전달력이 강화됨.
- **공고 리스트:** 포스터 가독성이 대폭 향상되었으며, 마감 임박순 정렬을 통해 정보 탐색 효율성을 확보함.
- **어드민 패널:** 불필요한 메뉴(팝업 관리)를 제거하고 공지사항 내에서 모든 팝업 설정을 완료할 수 있는 통합 워크플로우를 완성함.
