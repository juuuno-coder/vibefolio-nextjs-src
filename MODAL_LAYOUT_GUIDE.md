```
┌─────────────────────────────────────────────────────────────┐
│  [X 닫기]                                                    │
│                                                              │
│  ┌──────────────────────┐  ┌────┐  ┌──────────────┐        │
│  │                      │  │    │  │              │        │
│  │                      │  │ 액  │  │   댓글 영역   │        │
│  │    상세 이미지        │  │ 션  │  │              │        │
│  │    (60% 너비)        │  │ 바  │  │   (18% 너비)  │        │
│  │                      │  │    │  │              │        │
│  │                      │  │48px│  │              │        │

```

## 주요 변경사항

1. **모달 너비**: 화면의 90% (max-width: 1400px)
2. **이미지 영역**: 60% 너비
3. **액션바**: 이미지 우측 48px 위치, 고정 배치
4. **댓글 영역**: 18% 너비, 우측 고정
5. **배경**: 반투명 검은색 (bg-black/80)

## 액션바 아이콘 순서 (위→아래)

1. 프로필 이미지
2. 제안하기 (Send)
3. 좋아요 (Heart)
4. 컬렉션 (Bookmark)
5. 공유하기 (Share2)
6. 댓글 (MessageCircle)
